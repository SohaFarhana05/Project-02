Data Analyst Agent API

AI-powered data analysis, visualization, and insights using LLMs

API Documentation
Endpoint
POST /api/
Required Files
  • questions.txt - Analysis questions/tasks
Optional Attachments
  • *.csv - Data files
  • *.png/jpg - Image files
  • Other data formats
Response Format

JSON array or object based on questions

Features
  • Web scraping & data sourcing
  • Data preparation & cleaning
  • Statistical analysis
  • Data visualization
  • Automatic error correction
  • Base64 image encoding
Example Usage
cURL Example
curl -X POST "/api/" \
  -F "questions.txt=@questions.txt" \
  -F "data.csv=@data.csv" \
  -F "image.png=@image.png"
